Les tests du \(\chi^2\)

Test d’ajustement

Dans les situations où on souhaite confronter des effectifs obtenus expérimentalement à des prévisions faites sur base d’hypothèses théoriques, et quand les effectfs en questions sont suffisamment grands (> 5, typiquement), on a souvent recourt à un test de \(\chi^2\).

Comme le montrent l’exemple suivant, le test de \(\chi^2\) d’ajustement permet de vérifier si les données récoltées pour un échantillon correspondent à une distribution théorique de probabilité \(P\) définie à priori.

Les hypothèses de ce test sont les suivantes :
  • H0 : les données observées suivent la loi théorique
  • H1 : les données observées ne suivent pas la loi théorique

Prenons l’exemple de la loi de Mendel :

Imaginons que deux lignées de souris, de génotypes respectifs AA et BB, sont croisées. Les produits du croisement (appelés F1 en génétique) seront tous de génotypes AB (ce qui signifie que, pour locus étudié, ils ont reçu une copie (appelée “allèle”) A d’un de leurs parents et une copie B de l’autre, ce qui est nécessairement le cas dans la situation décrite ici de croisement de lignées AA et BB). A la génération suivante, on croise des individus F1 (c’est-à-dire de génotypes AB) entre eux de manière aléatoire, pour générer des individus de seconde génération (notés F2). Sous l’hypothèse nulle où chaque individu F1 croisé a une probabilité de 50 % de transmettre l’allèle A et une probabilité de 50 % de transmettre l’allèle B, on peut déduire que les probabilités qu’un individu F2 ait un génotype AA, AB ou BB sont respectivement de 25 %, 50 % et 25 %.

📊 Cours : Le test du Chi² d’ajustement

1. Définition générale

Le test du Chi² d’ajustement (ou test de conformité) permet de vérifier si la distribution d’un échantillon observé correspond à une distribution théorique attendue. Il s’agit d’un test statistique non paramétrique, souvent utilisé pour des données catégorielles (qualitatives ou discrétisées).

2. Objectifs du test

  • Comparer une distribution observée à une distribution théorique connue.
  • Tester l’hypothèse que les fréquences observées suivent une loi théorique (comme la loi uniforme, binomiale, de Poisson, etc.)

3. Hypothèses

  • H₀ (hypothèse nulle) : les données suivent la distribution théorique.
  • H₁ (hypothèse alternative) : les données ne suivent pas cette distribution.

4. Formule du test du Chi²

La statistique du test est :

[ ^2 = _{i=1}^{k} ]

où :

  • ( O_i ) = fréquence observée pour la modalité ( i ),
  • ( E_i ) = fréquence attendue (selon la loi théorique) pour la modalité ( i ),
  • ( k ) = nombre de modalités (ou classes).

5. Conditions d’application

  • Les observations doivent être indépendantes.
  • L’effectif total doit être suffisamment grand (souvent ( n > 30 )).
  • Les fréquences théoriques ( E_i ) doivent être supérieures ou égales à 5 pour chaque modalité (ou regroupement si nécessaire).

6. Degrés de liberté

[ = k - 1 - p ]

  • ( k ) = nombre de modalités (ou classes),
  • ( p ) = nombre de paramètres estimés à partir des données (ex : moyenne ou variance si la loi théorique n’est pas entièrement connue).

7. Étapes du test

  1. Formuler les hypothèses H₀ et H₁.
  2. Calculer les fréquences observées ( O_i ).
  3. Calculer les fréquences attendues ( E_i ) en fonction de la distribution théorique.
  4. Calculer la statistique du test ( ^2 ).
  5. Déterminer la valeur critique (ou p-value) à partir de la loi du ( ^2 ) avec les degrés de liberté.
  6. Conclusion : si ( ^2_{} > ^2_{} ), on rejette H₀.

9. Interprétation

  • Si on rejette H₀, cela signifie que les écarts observés entre les fréquences observées et attendues sont trop grands pour être dus au hasard.
  • Si on ne rejette pas H₀, les écarts peuvent être dus à la variabilité aléatoire.

10. Applications courantes

  • Étude de conformité d’un jeu de dés (uniformité).
  • Vérification de la loi de Poisson (ex : nombre d’appels téléphoniques par minute).
  • Analyse de répartition démographique (sexe, âge, catégories socio-pro).
